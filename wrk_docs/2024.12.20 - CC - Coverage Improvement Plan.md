## Implementation Notes

### Mock Infrastructure Required

1. **MockTurnServer**
   - Responds to ALLOCATE_REQUEST
   - Sends ALLOCATE_RESPONSE/ALLOCATE_ERROR
   - Handles REFRESH_REQUEST
   - Handles CREATE_PERMISSION_REQUEST
   - Sends DATA_INDICATION

2. **MockStunServer**
   - Responds to BINDING_REQUEST
   - Sends BINDING_RESPONSE with XOR-MAPPED-ADDRESS

3. **TestCertificates**
   - Self-signed CA certificate
   - Server certificate signed by CA
   - Client certificate (optional)

### Testing Patterns

1. **Async Mock Pattern:**
   ```rust
   async fn test_with_mock_server() {
       let mock = MockServer::start().await;
       let client = Client::new(mock.addr());
       // test interaction
   }
   ```

2. **Response Injection Pattern:**
   ```rust
   fn build_mock_response(txn_id: &[u8]) -> Bytes {
       // Build valid STUN/TURN response
   }
   ```

3. **Timeout Testing Pattern:**
   ```rust
   #[tokio::test]
   async fn test_timeout() {
       let result = tokio::time::timeout(
           Duration::from_millis(100),
           long_operation()
       ).await;
       assert!(result.is_err());
   }
   ```
