## Critical Coverage Gaps

### 1. mdsiprtp-ice/src/turn.rs (63.13%)
**Missing Coverage:** 469 lines

The TURN client implementation has unit tests for helper functions but lacks tests for the core async methods:
- `TurnClient::allocate()` - Initial TURN allocation
- `TurnClient::refresh()` - Allocation refresh
- `TurnClient::create_permission()` - Permission creation for peers
- `TurnClient::send_data()` - Data relay through TURN
- `TurnClient::receive_data()` - Receiving relayed data
- Response parsing functions for actual TURN responses

**Root Cause:** These functions require a TURN server or comprehensive mocking.

### 2. mdsiprtp-transport/src/tls.rs (63.57%)
**Missing Coverage:** 247 lines

TLS transport has helper function tests but lacks coverage for:
- `TlsConnectionState::read_message()` - Reading complete SIP messages
- `TlsConnectionState::try_parse_message()` - Message framing
- `TlsTransport::run_client()` - Client connection loop
- `TlsTransport::run_server()` - Server accept loop
- Actual TLS handshake and data transfer

**Root Cause:** Requires TLS server setup with certificates or mocking.

### 3. mdsiprtp-ice/src/agent.rs (69.88%)
**Missing Coverage:** 241 lines

ICE agent tests cover state management but miss:
- `perform_connectivity_check()` - STUN binding request/response
- STUN attribute validation in checks
- Pair nomination with USE-CANDIDATE
- ICE restart functionality
- Server reflexive candidate gathering

**Root Cause:** Requires STUN server interaction or mocking.

### 4. mdsiprtp-transport/src/resolver.rs (78.17%)
**Missing Coverage:** 110 lines

DNS resolver tests use IP addresses to bypass DNS. Missing:
- SRV record lookup (`lookup_srv()`)
- NAPTR record handling (`lookup_naptr()`)
- DNS failure handling
- SRV weight/priority sorting

**Root Cause:** DNS queries require network or mock DNS server.

### 5. mdsiprtp-ice/src/stun.rs (89.60%)
**Missing Coverage:** 49 lines

STUN module is mostly covered but missing:
- Error response parsing
- Unknown attribute handling
- Malformed response handling

### 6. mdsiprtp-transport/src/tcp.rs (90.96%)
**Missing Coverage:** 45 lines

TCP transport mostly covered but missing:

### 7. mdsiprtp-sip/src/message.rs (96.19%)
