# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc] as its code of
conduct.

The full text — including the standards, scope, and enforcement guidelines —
is canonical at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

To report behaviour that you believe violates this code of conduct, contact
the project maintainer privately at **0x4d444d44@gmail.com**, or open a
private [GitHub Security Advisory](https://github.com/0x4D44/rsiprtp/security/advisories/new)
(GitHub allows non-security reports via this channel too) if you'd rather
keep it on-platform. Reports are handled in confidence and on a best-effort
basis — see [SECURITY.md](SECURITY.md) for context on the level of
maintenance this project receives.

For security vulnerabilities (not conduct issues), see
[SECURITY.md](SECURITY.md) for the private disclosure channel.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
