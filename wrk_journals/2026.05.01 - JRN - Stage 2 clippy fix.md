## Fixes

### 1+2. `is_multiple_of(4)` → `% 4 != 0` (negated, since the original used `!`)

`crates/rsiprtp/src/rtp/rtcp.rs:559` and `:668`:

```rust
// Before
while !buf.len().is_multiple_of(4) {
    buf.put_u8(0);
}

// After
while buf.len() % 4 != 0 {
    buf.put_u8(0);
}
```

Behaviour-equivalent. Both call sites pad an RTCP buffer up to a 32-bit
boundary, exactly the same trigger condition as before.

### 3. `Duration::abs_diff` (1.81) → manual branch

`crates/rsiprtp/src/rtp/session.rs:222–227`:

```rust
// Before
let diff = self.smoothed_rtt.abs_diff(rtt);

// After
let diff = if self.smoothed_rtt > rtt {
    self.smoothed_rtt - rtt
} else {
    rtt - self.smoothed_rtt
};
```

Same semantics: returns `|self.smoothed_rtt − rtt|` as a `Duration`. The branch
on `>` (strict) routes the `==` case through the `else` arm, which yields
`Duration::ZERO` — same as `abs_diff` would.

### 4. `Box::new(G722Codec::new())` → `Box::default()`

`crates/rsiprtp/src/session/session_codec.rs:51`:

```rust
// Before
"g722" => Ok(SessionCodec::G722(Box::new(G722Codec::new()))),

// After
"g722" => Ok(SessionCodec::G722(Box::default())),
```

`G722Codec` already has `impl Default` at `crates/rsiprtp/src/media/codec/g722.rs:97`.
Verified `G722Codec::default()` calls `G722Codec::new()` (or is functionally
identical), so the produced codec is the same.
