# Security

`rsiprtp` is a personal project, not a supported product — no SLA, no
security team, no guarantee a report results in a fix. If you're
shipping this in something that matters, plan on maintaining your own
fork.

If you find a vulnerability and want to share it privately rather than
in a public issue, please use
[GitHub Security Advisories](https://github.com/0x4D44/rsiprtp/security/advisories/new).
A description of the impact, a proof-of-concept, and the affected commit
are all helpful. I'll respond when I can — no promises on timing.
